# Part 1 – Farmer

**Owner:** Philip

## User Stories

- As a farmer, I want to upload maize images, so that I can analyse crop conditions.
- As a farmer, I want the system to automatically count maize tassels, so that I do not need to perform manual counting.
- As a farmer, I want to view counting results clearly, so that I can understand plant growth.
- As a farmer, I want to see highlighted tassels on images, so that I can visually verify the results.
- As a farmer, I want to upload multiple images at once, so that I can save time.
- As a farmer, I want to receive results within a short response time, so that I can make timely decisions.
- As a farmer, I want to access the system via mobile devices, so that I can use it in the field.
- As a farmer, I want an intuitive and user-friendly interface, so that I can use the system easily.
