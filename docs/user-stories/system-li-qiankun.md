# Part 5 – System

**Owner:** Li Qiankun

## User Stories

- As a system, I want to preprocess image data, so that the model can achieve better performance.
- As a system, I want to train deep learning models, so that tassel counting accuracy can be improved.
- As a system, I want to evaluate model performance using appropriate metrics, so that accuracy can be measured and improved.
- As a system, I want to deploy the trained model as a service, so that users can access it online.
- As a system, I want to support system updates and model improvements, so that new features and enhancements can be integrated.
