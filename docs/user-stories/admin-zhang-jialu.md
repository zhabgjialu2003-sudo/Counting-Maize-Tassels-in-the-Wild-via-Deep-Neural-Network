# Part 4 – Admin

**Owner:** Zhang Jialu

## User Stories

- As an admin, I want to manage user accounts, so that I can control system access.
- As an admin, I want to store uploaded images securely, so that data is protected.
- As an admin, I want to monitor system usage, so that I can ensure system performance.
- As an admin, I want to manage datasets, so that the system maintains high-quality training data.
- As an admin, I want to control user permissions, so that different roles have appropriate access levels.
- As an admin, I want to back up data regularly, so that data loss is prevented.
