# Part 3 – Agronomist

**Owner:** Zhang YiXin

## User Stories

- As an agronomist, I want to evaluate plant health based on tassel count, so that I can provide recommendations.
- As an agronomist, I want to monitor crop growth over time, so that I can track development stages.
- As an agronomist, I want to detect abnormal patterns in tassel counts, so that I can identify potential issues early.
- As an agronomist, I want a dashboard view of multiple fields, so that I can analyse large-scale crop conditions.
- As an agronomist, I want summarized insights, so that I can make decisions efficiently.
